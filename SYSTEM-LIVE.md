## EMERGENCY CONTACTS

If issues arise:

1. **Firecrawl Authentication Fails**
   - Run: `firecrawl login` (manual re-authentication)
   - Fallback: HTTP-based collection (slower but functional)

2. **Report Generation Fails**
   - Check: `.agents/skills/insurance-market-intelligence/` exists
   - Verify: `.agents/agents/insurance-broker-specialist.md` is loaded
   - Manual: Run `.agents/scripts/market_intel_weekly.ps1` directly

3. **Routing Not Working**
   - Verify trigger keywords in user message match list
   - Check: `.agents/rules/GEMINI.md` for override rules
   - Restart conversation (cache refresh)

4. **Credit Exhaustion**
   - Monitor: `firecrawl --status` (shows remaining credits)
   - Escalate: Finance team for credit refresh
   - Threshold: Alert sent at <500 credits
